# OnmicBot

A bot for interacting with Trello boards from Telegram.


### Installing


Download or clone this git repo

When in folder run:
```
npm install
npm start / dev
```

Set access and tokens in the `.env`

```
TRELLO_KEY = ""
TRELLO_TOKEN = ""
TELEBOT_TOKEN = ""
TELECHAT_ID =""
BOARD_NAME = ""
BOARD_ID = ""
ID_LABELS = ""
ID_MEMEBERS = ""
```

### Command

```
/hello
/help: for help
/bug: for log bug and create card in trello
```
